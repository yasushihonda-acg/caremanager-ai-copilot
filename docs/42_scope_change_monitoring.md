
# 42_scope_change_monitoring.md - Scope Change: Removal of Monitoring Feature

## 1. 概要
*   **決定日:** 2025年12月03日
*   **決定事項:** 本デモアプリケーション（v1.2.x）において、**「モニタリング機能」を実装範囲から除外（削除）**する。
*   **理由:** デモのスコープを「アセスメントからケアプラン作成（ドラフト）」のフローに集中させるため。

## 2. 影響範囲調査 (Impact Analysis)

### A. 削除対象コンポーネント
*   `MonitoringView`: 月次ログの表示・入力機能。
*   `App.tsx` 内のタブナビゲーションおよびレンダリングロジック。

### B. データ構造への影響
*   `types.ts` 内の `MonitoringLog` インターフェースは不要となる。
*   `CarePlan` や `AssessmentData` への依存関係はないため、他の機能（アセスメント、プラン作成）は正常に動作する。

### C. 既存コードの修正方針
1.  **`App.tsx`**:
    *   `MonitoringView` のインポートを削除。
    *   `monitoringLogs` Stateおよび初期データを削除。
    *   タブナビゲーションの配列から「モニタリング」を削除。
    *   条件付きレンダリングブロックを削除。
2.  **`types.ts`**:
    *   `MonitoringLog` 型定義を削除。

## 3. 結論
本変更による他機能への副作用（リグレッション）はないと判断し、即時適用を行う。
