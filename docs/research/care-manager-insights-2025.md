```yaml
  - 課題整理総括表

  - 第1表: 居宅サービス計画書(1) - 総合方針
  - 第2表: 居宅サービス計画書(2) - ニーズ・目標・サービス
  - 第3表: 週間サービス計画表
  - 第4表: サービス担当者会議記録
  - 第5表: 支援経過記録

  - 第7表: サービス利用票別表

その他:
  - モニタリングシート（月1回）
  - 入院時情報連携シート
  - 退院・退所加算様式
  - 主治医連絡票
```

```
[インプット]                    [処理]                      [アウトプット]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

                       └──▶ 課題分析・優先度付け ──▶ 第1表（総合方針）

                                              │

```

### 3.3 第2表の構造（AI生成に最適）

| 項目 | 内容 | AI支援の可能性 |
|------|------|---------------|
| **生活全般の解決すべき課題（ニーズ）** | 利用者の具体的な課題 | ⭐⭐⭐ アセスメントから自動抽出 |
| **長期目標** | 6ヶ月〜1年で達成する目標 | ⭐⭐⭐ ニーズに応じた文例提案 |
| **短期目標** | 長期目標達成のためのステップ | ⭐⭐⭐ 段階的目標の自動生成 |
| **サービス内容** | 具体的な支援内容 | ⭐⭐ サービス種別に応じた提案 |
| **サービス種別** | デイサービス、ヘルパー等 | ⭐⭐ ニーズとのマッチング |
| **頻度・期間** | 週2回、3ヶ月間など | ⭐⭐ 標準的な頻度の提案 |

## 5. 制度改正の動向（2025-2027）

### 5.1 2024年度介護報酬改定の主要変更

```yaml
改定率: +1.59%
  - 介護職員処遇改善: +0.98%
  - その他: +0.61%（ケアマネ含む）

居宅介護支援の主な変更:
  1. 担当利用者数の見直し:
     - 逓減制緩和: 40人未満 → 45人未満

  2. 特定事業所加算の引き上げ:
     - 一律14単位増加
     - 要件追加: ヤングケアラー等の研修参加

  3. オンラインモニタリング解禁:
     - 条件: 2ヶ月に1回は訪問必須

  4. 入院時情報連携加算の拡充:
     - (Ⅰ): 200→250単位
     - (Ⅱ): 100→200単位
```

### 5.2 2027年改正に向けた検討事項

| 検討項目 | 現状 | 改正方向 |
|---------|------|---------|
| **更新制度** | 5年ごとに研修受講必須 | 廃止の方向（研修は義務継続） |
| **受験資格** | 実務経験5年 | 3年に短縮検討 |
| **法定研修** | 初回88時間、更新32時間 | 分割受講、時間短縮検討 |
| **処遇改善加算** | 居宅介護支援は対象外 | 2026年度期中改定で対象追加予定 |
| **受験資格拡大** | 特定の国家資格保持者 | 公認心理師等を追加 |

## 8. 用語集

| 用語 | 説明 |
|------|------|
| **アセスメント** | 利用者の状態・ニーズを評価・分析すること |
