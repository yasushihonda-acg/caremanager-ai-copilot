
# 46_scope_change_hide_careplan.md - Scope Adjustment: Temporarily Hide Care Plan

## 1. 概要
*   **決定日:** 2025年12月03日
*   **変更内容:** 「ケアプラン作成機能（第1表・第2表）」へのアクセスをUI上で一時的に非表示にする。
*   **目的:** デモのフォーカスを「スマート・アセスメント」機能に一点集中させるため。

## 2. 影響範囲調査 (Impact Analysis)

### A. 安全性確認 (Golden Thread Check)
*   **構造:** `Assessment` (Upstream) -> `CarePlan` (Downstream)
*   **判定:** 下流工程であるケアプランを隠しても、上流のアセスメント機能、AI解析、法的整合性チェックのロジック自体には悪影響を及ぼさない。**安全である。**

### B. 対応方針
*   **削除ではなく非表示:** 将来的な復活を前提とするため、`types.ts` や `services/` 内のロジックは削除しない。
*   **App.tsx:** タブナビゲーションの配列から `plan` を除外するのみとする。これにより、ユーザーはアセスメント画面のみを操作することになる。
